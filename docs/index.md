# Welcome to the Monorepo Documentation

This is a template documentation site powered by [Vitepress](https://vitepress.dev/).

## Getting Started

1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Start the development server:
   ```bash
   pnpm run docs:dev
   ```

## Structure

The documentation system automatically discovers packages in the `packages/` directory and syncs their `README.md` and `CHANGELOG.md` files.
